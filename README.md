# Grab'it

A free, open-source video downloader for Android. No ads, no limits.

Download videos from YouTube, Instagram, TikTok, Facebook, Twitter/X and 1000+ sites powered by yt-dlp.

## Features

- **Multi-source** -- YouTube, Instagram, TikTok, Facebook, Twitter/X, and more
- **Format picker** -- choose video quality, resolution, or audio-only (MP3)
- **Playlist support** -- download entire YouTube playlists
- **Quick mode** -- share a link, download starts instantly in best quality
- **Built-in browser** -- browse and download without leaving the app
- **Background downloads** -- queue multiple downloads, pause/resume anytime
- **Video player** -- stream or play downloads with Picture-in-Picture
- **Subtitles** -- download and embed subtitles in your preferred language
- **Safe Zone** -- hide downloads behind biometric authentication
- **App lock** -- require fingerprint/PIN to open the app
- **Clipboard detect** -- auto-detects video links you copy
- **WiFi-only mode** -- restrict downloads to WiFi connections
- **Auto-update** -- app and yt-dlp engine update themselves silently
- **Deep links** -- tap a video URL and Grab'it opens directly
- **Share target** -- share from any app to start downloading
- **Dark/Light theme** -- follows your preference

## Install

Download the latest APK from [Releases](https://github.com/fless-lab/Grab-it/releases/latest).

## Build

```bash
git clone https://github.com/fless-lab/Grab-it.git
cd Grab-it
./gradlew assembleDebug
```

Requires JDK 17 and Android SDK 34.
